# 🦞 Lobster Desktop Widget

デスクトップ常駐型のサブエージェント監視ウィジェット。
Pixoo-64電光掲示板のデスクトップ版 — 最大8匹のロブスターが同時に活動状況を表示。

## ✨ 特徴

- 🦞 **最大8匹同時表示** — 各AIキャラが個別にアニメーション
- ⏱️ **リアルタイムタイマー** — 各サブエージェントの経過時間（虹色サイクリング）
- 📊 **カウント表示** — 稼働中エージェント数（1=緑→7=赤グラデーション）
- 📜 **スクロールテキスト** — 最新タスク名 or 最優先タスク（todo-priority.md連携）
- 💤 **スリープモード** — 10分アイドルで寝るロブちゃん（ZZZ + よだれ）
- 🖥️ **デスクトップ常駐** — 透過背景、常に最前面、ドラッグ移動可能
